# Brobillet Bibliotek / Ticket System Library
Når man kører over Storebæltsbroen og Øresundsbroen, skal man altid betale for at kunne krydse broerne via en afgift eller en billet. Dette system implementerer en standardløsning gennem et _library_ der kan bruges af de to broer til at opkræve betaling fra biler og motorcykler. Biblioteket kan skræddersyres, hvis firmaet ønsker at implementere flere funktionaliteter. Dette sker via et DLL til videregående arbejde. Til slut testes alt via unit testing.

Dette bibliotek blev skrevet i forbindelse med en obligatorisk opgave på tredje semester, og alt er skrevet i C# med brug af Visual Studio.
